# Lobby

Lobby is a mobile-first social messaging application built for gamers. It provides a clean, focused environment where players can communicate, form squads, share gameplay moments, and organize group activities without the clutter or complexity of traditional gaming communication platforms.

The goal of Lobby is to combine the simplicity of messaging apps like WhatsApp with features designed specifically for gaming communities.

---

## Overview

Lobby allows gamers to:

- Chat with friends in real time
- Create private groups or gaming communities
- Share gameplay clips, screenshots, and updates
- See what games friends are currently playing
- Organize squads and party chats quickly
- Invite friends directly into gaming sessions

The design emphasizes clarity, usability, and performance. The interface avoids flashy visual effects and instead uses a clean, professional layout suitable for a production application.

---

## Core Features

### Real-Time Messaging
Users can send and receive messages instantly in one-on-one conversations or group chats. Messages support text, images, short clips, and voice notes.

### Friends System
Users can add friends, view their activity status, and see which game they are currently playing.

### Group Chats and Communities
Lobby allows players to create group chats for squads, teams, or gaming communities. Each group can have multiple channels for discussions such as strategy, general chat, or sharing clips.

### Game Activity Status
Friends can see whether someone is online, offline, or currently in a game.

### Party and Game Invites
Users can quickly invite friends to join a game session or voice party.

### Media Sharing
Players can share screenshots, gameplay clips, and images directly inside chats.

---

## Application Structure

The application is organized into several main screens.

### Chats
Displays recent conversations with message previews, timestamps, and unread indicators.

### Friends
Shows a list of gamer contacts along with their activity status and current game.

### Groups
Lists communities or squad groups with recent activity and channel navigation.

### Profile
Displays the user's gamer tag, avatar, favorite games, and achievements.

---

## Design Philosophy

Lobby focuses on:

- Clean and minimal user interface
- Neutral color palette with subtle accents
- Clear typography and spacing
- Mobile-first usability
- Familiar messaging patterns to reduce learning curve

The design avoids neon or cyberpunk aesthetics and instead aims for a professional, realistic application interface.
